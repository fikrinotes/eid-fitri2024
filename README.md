# Eid Al-Fitr 1445H Digital Greeting 🌙

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://fikrinotes.github.io/eid-fitri2024/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Sebuah aplikasi web sederhana untuk mengirimkan ucapan selamat Idul Fitri 1445 H secara digital. Proyek ini dirancang agar ringan, responsif, dan memberikan pengalaman interaktif bagi penerimanya.

## ✨ Fitur Utama
* **Interaktif & Responsif**: Tampilan yang menyesuaikan dengan perangkat mobile maupun desktop.
* **Manajemen Pesan**: Pengiriman pesan antar pengguna hanya dengan memanfaatkan teknologi frontend.
* **Custom Message**: Pesan ucapan yang dipersonalisasi.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript, Crypto.js
* **Font**: FontAwesome, Google Font
* **Deployment**: GitHub Pages
